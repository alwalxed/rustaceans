# The Rust Book (Rustaceans)

Collaborative slide decks for the Rust Book study group.
Each chapter has its own folder; each presenter has a separate `.md` file.

## Presenters
alwaleed · joudy · mohammed · lina · waleed

chapters/

## Setup & Run
```bash
# install deps
npm install

# start server
npm run dev
```
